# Factory Four Status Page

## Setup

Clone the project

```
git clone git@github.com:alexisibarra/inclusion-take-home-challenge.git
```

Use yarn to install all dependencies

```bash
yarn install
```

Set the environment variables

```bash
cp env-example .env
```

Update .env with the proper data for the env variables

Optionally start the app

```bash
yarn start
```

The amount of seconds the app delays to fetch the status every time can be
changed in

```
/src/config/INTERVAL_SECONDS.tsx
```
